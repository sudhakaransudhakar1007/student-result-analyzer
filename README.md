# student-result-analyzer

This is a simple Python mini project.

## Features
- Takes student marks as input
- Checks Pass / Fail
- Counts total Pass and Fail students

## Concepts Used
- Functions
- For loop
- If-else
- List
- User input

## How to Run
1. Open the file `student_result_analyzer.py`
2. Run using Python
3. Enter number of students and marks
